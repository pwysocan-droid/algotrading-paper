# algotrading-paper VPS crontab (trader user)
# Install with: crontab -e  →  paste the line below  →  save.
# Verify with:  crontab -l
#
# Every 5 minutes: run the full surface pipeline. Master log captures
# stdout/stderr; cron-fetch.sh also writes a dated per-day log.
*/5 * * * * /home/trader/algotrading-paper/vps/cron-fetch.sh >> /home/trader/algotrading-paper/vps/logs/cron-master.log 2>&1

# Nightly at 03:32 UTC: adversarial skeptic (adversarial_cron.py).
# 03:32 sits between the 03:30 and 03:35 fetch cycles so the two jobs
# never race each other's git operations.
32 3 * * * /home/trader/algotrading-paper/vps/cron-skeptic.sh >> /home/trader/algotrading-paper/vps/logs/cron-master.log 2>&1

# Fridays at 03:34 UTC: full 4G bear-case review (adversarial_cron.py
# --friday). Two minutes after the skeptic slot so the jobs never
# overlap; a missed Friday review is a Phase 1b archive trigger
# (phase1-review.md § 5), so this runs from cron, not operator memory.
34 3 * * 5 /home/trader/algotrading-paper/vps/cron-friday.sh >> /home/trader/algotrading-paper/vps/logs/cron-master.log 2>&1

# Foundry autopilot twice daily (04:02, 16:02 UTC): advances the
# gauntlet/synthesis conditions of the closed seam. flock-guarded.
2 4,16 * * * /home/trader/algotrading-paper/vps/cron-foundry.sh >> /home/trader/algotrading-paper/vps/logs/cron-master.log 2>&1
