# The Service — falsification registry (CONSTITUTION Art. 3, 4)

Public, timestamped autopsy registry. Each autopsy is pre-registered as
`prereg-<id>.md` (target claim, data, method, cost model, kill criterion)
BEFORE any result exists (3.1). Before publication: (1) the mechanical
conflict declaration is attached via scripts/conflict_check.py (4.2);
(2) the blind-adversary stage runs (3.5). Results publish regardless of
outcome — survivals as prominently as kills (3.1).
